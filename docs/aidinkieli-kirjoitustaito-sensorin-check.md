# Sensorin silmin – Kirjoitustaidon kokeen arviointi

> Tämä dokumentti katsoo tekstiäsi **sensorin eli tarkastajan silmin**.
> Kun ymmärrät mitä sensori etsii, osaat kirjoittaa tekstin joka vastaa odotuksiin.

---

## Sensorin 5 linssiä: NÄKRA-muistisääntö

Sensori lukee tekstisi viiden linssin läpi. Muista **NÄKRA**:

```
╔══════════════════════════════════════════════════════════════╗
║                                                              ║
║     N Ä K R A                                                ║
║     │ │ │ │ │                                                ║
║     │ │ │ │ └─── A ineistojen käyttö                         ║
║     │ │ │ └───── R akenne                                    ║
║     │ │ └─────── K ieli ja tyyli                             ║
║     │ └───────── Ä äni ja näkökulma                          ║
║     └─────────── N äkökulma- ja sisältövalinnat              ║
║                                                              ║
║     Painotus:  ███████████  Kokonaiskuva                     ║
║                ██████████   Rakenne                          ║
║                ██████████   Kieli ja tyyli                   ║
║                ███████      Aineistot                        ║
║                ███████      Näkökulma ja sisältö             ║
║                                                              ║
╚══════════════════════════════════════════════════════════════╝
```

**Kokonaiskuva, rakenne sekä kieli ja tyyli painavat eniten.**

## Linssi 1: NÄKÖKULMA JA SISÄLTÖ – "Mitä sanot?"

### Mitä sensori kysyy itseltään:

```
┌─────────────────────────────────────────────────────┐
│  - Onko näkökulma RAJATTU ja SELKEÄ?                │
│  - Vastaako teksti VALITTUUN AIHEESEEN?             │
│  - Onko tekstissä OMAA AJATTELUA?                   │
│  - Onko pohdinta MONIPUOLISTA (ei yksipuolista)?    │
│  - Käsitelläänkö aihetta SYVÄLLISESTI?              │
└─────────────────────────────────────────────────────┘
```

### Pisteportaat:

| Taso | Mitä sensori näkee |
|------|--------------------|
| 0–10 p | Näkökulmaa ei ole tai se ei liity aiheeseen. Kokelas ei vastaa tehtävään. |
| 20 p | Näkökulma on epämääräinen. Pohdinta pinnallista tai kaavamaista. |
| 30 p | Näkökulma on tunnistettavissa. Väitteitä yritetään perustella. Pohdintaa on, mutta sitä on niukasti tai yksipuolisesti. |
| 40 p | Näkökulma on **selkeä ja rajattu**. Pohdinta on monipuolista ja perustelut vakuuttavia. |
| 50–60 p | Näkökulma on **oivaltava ja kiinnostava**. Kirjoittaja osoittaa itsenäistä ajattelua, käsittelee aihetta usealta kantilta ja yllättää lukijan. |

### Sensorin vinkki (suora lainaus):
> *"Kaikki kirjoitustaidon kokeen aiheet edellyttävät kokelaan omaa rajausta. Näkökulman valinta on yksi arviointikriteereistä."*
>
> *"Kokelaiden tekstejä ei vertailla toisiinsa, vaan jokainen vastaus arvioidaan suhteessa arviointikriteereihin."*

### Mitä sinun pitää tehdä:
```
 KYLLÄ                              EI
 ──────                             ──────
 ✓ Rajaa aihe tarkasti              ✗ "Tekoäly on monimutkainen asia"
   "Tekoäly uhkaa luovuutta"           (liian laaja, ei ole kanta)

 ✓ Ota selkeä kanta                 ✗ Luettele faktoja ilman kantaa
   ja perustele se

 ✓ Käsittele myös vastanäkökulmia   ✗ Kirjoita vain yhdestä
   ("toisaalta...")                    näkökulmasta

 ✓ Pohdi MIKSI, ei vain MITÄ        ✗ Kuvaa vain ilmiötä pintatasolla
```

## Linssi 2: RAKENNE – "Miten jäsennät?"

### Mitä sensori kysyy itseltään:

```
┌─────────────────────────────────────────────────────┐
│  * Onko tekstillä OTSIKKO?                          │
│  * Onko ALOITUS kiinnostava ja johdatteleva?        │
│  * Onko KAPPALEJAKO LOOGINEN?                       │
│  * Eteneekö teksti JOHDONMUKAISESTI?                │
│  * Onko LOPETUS kokoava (ei vain "loppuu")?         │
│  * Pysyykö teksti KASASSA vai rönsyileekö?          │
└─────────────────────────────────────────────────────┘
```

### Pisteportaat:

| Taso | Mitä sensori näkee |
|------|--------------------|
| 0–10 p | Teksti on jäsentymätön. Kappaleiden välillä ei ole yhteyttä. |
| 20 p | Kappaleet eivät kytkeydy toisiinsa. Siirtymät puuttuvat tai ovat äkkinäisiä. |
| 30 p | Teksti muodostaa kokonaisuuden, mutta rakenteessa on paikoin puutteita. |
| 40 p | Otsikko toimii, aloitus ja lopetus tukevat tavoitetta. **Kappaleet jäsennettyjä ja sidoksissa toisiinsa.** |
| 50–60 p | Rakenne on **eheä ja tukee argumentointia**. Siirtymät ovat luontevia. Teksti tuntuu yhtenäiseltä kokonaisuudelta. |

### Sensorin vinkki:
> *"Otsikko on tärkeä, koska se kertoo tekstisi näkökulman. Jos se on selkeä muutenkin, ei otsikon puuttuminen ole kohtalokasta – mutta älä jätä sitä pois."*
>
> *"On tavallista, että yli 9000 merkin teksteissä rakenne ei pysy ehyenä – toistoa, rönsyilyä."*

### Ideaalirakenne – sensorin silmin:

```
┌──────────────────────────────────────────────────────────┐
│  OTSIKKO                                                 │
│  "Tekoäly voi koitua luovuuden tuhoksi"                  │
│  → Kertoo näkökulman, houkuttelee lukemaan               │
├──────────────────────────────────────────────────────────┤
│  JOHDANTO (1 kappale)                                    │
│  • Herätä kiinnostus                                     │
│  • Esittele aihe ja näkökulma                            │
│  • Kerro, mistä kulmasta käsittelet                      │
├──────────────────────────────────────────────────────────┤
│  KÄSITTELY (3–5 kappaletta)                              │
│  • Kappale = yksi ajatus/argumentti                      │
│  • Jokaisessa: väite → perustelu → esimerkki/aineisto    │
│  • Sidossanat kappaleiden välissä                        │
│  • Vastanäkökulma mukaan ("toisaalta...")                │
├──────────────────────────────────────────────────────────┤
│  LOPETUS (1 kappale)                                     │
│  • Kokoa pääargumentit yhteen                            │
│  • Älä tuo uutta asiaa                                   │
│  • Avoin kysymys tai tulevaisuuteen katsova päätös       │
└──────────────────────────────────────────────────────────┘
```

## Linssi 3: KIELI JA TYYLI – "Miten kirjoitat?"

### Mitä sensori kysyy itseltään:

```
┌─────────────────────────────────────────────────────┐
│  * Ovatko VIRKKEET selkeitä ja vaihtelevia?         │
│  * Onko SANASTO monipuolista ja tarkkaa?            │
│  * Hallitseeko kokelas KIRJAKIELEN NORMIT?          │
│  * Onko tyyli JOHDONMUKAINEN (ei pompi rekisteristä │
│    toiseen)?                                         │
│  * Välittyykö VIESTI lukijalle?                     │
└─────────────────────────────────────────────────────┘
```

### Pisteportaat:

| Taso | Mitä sensori näkee |
|------|--------------------|
| 0–10 p | Virke- ja lauserakenteissa pahoja puutteita. Viesti ei välity. |
| 20 p | Tyylissä ja sananvalinnoissa haparointia. Yleiskielen hallinnassa horjuvuutta. |
| 30 p | Kirjakielen normit kohtuullisesti hallinnassa. Lause- ja virkerakenteet pääosin kunnossa. |
| 40 p | Sananvalinnat **tarkoituksenmukaisia**. Kirjakielen normit **hyvin hallinnassa**. |
| 50–60 p | Kieli on **ilmaisuvoimaista, rikasta ja tarkkaa**. Tyyli palvelee tekstin tavoitetta. |

### Sensorin vinkki:
> *"Yksittäiset kirjoitusvirheet eivät haittaa. Runsaat virheet, jotka vaikuttavat luettavuuteen, vaikuttavat arviointiin."*
>
> *"Jos oikeinkirjoituksessa on virheitä mutta sanasto on rikasta ja virkkeet selkeitä, voi kokeesta saada silti ihan hyvät pisteet."*
>
> *"Oikeakielisyyttä arvioidaan sitä kriittisemmin, mitä korkeampia pisteitä tavoittelee."*

### Yleisimmät kielimokat:

```
 VIRHETYYPPI              ESIMERKKI                FIX
 ─────────────────────────────────────────────────────────
 Yhdyssanavirhe           teko äly → tekoäly       Liitä yhteen
 Pilkkuvirhe              Koska, hän → Koska hän   Ei pilkkua ennen
                          hän sanoi että →          konjunktiota (päälause),
                          hän sanoi, että           pilkku ENNEN sivulauseen
                                                    aloittavaa sanaa

 Toisto                   "asia on asia"            Käytä synonyymejä
 Puhekielisyys            "mä", "sit", "niinku"     → minä, sitten, ikään kuin
 Liian pitkät virkkeet    60-sanainen virke         Jaa kahdeksi virkkeeksi
 Liian lyhyet virkkeet    "Näin on. Se on totta."   Yhdistele sidossanoilla
```

## Linssi 4: AINEISTOJEN KÄYTTÖ – "Miten hyödynnät?"

### Mitä sensori kysyy itseltään:

```
┌─────────────────────────────────────────────────────┐
│  * Viittaako kokelas VÄHINTÄÄN KAHTEEN aineistoon?  │
│  * Onko aineisto INTEGROITU tekstiin luontevasti?   │
│  * Käykö kokelas DIALOGIA aineistojen kanssa?       │
│  * Onko aineisto TUKI omalle ajattelulle (ei        │
│    pelkkää referointia)?                            │
│  * Onko aineistoja tulkittu OIKEIN?                 │
└─────────────────────────────────────────────────────┘
```

### Pisteportaat:

| Taso | Mitä sensori näkee |
|------|--------------------|
| 0–10 p | Aineistoviittaukset puuttuvat tai aineistoja vain kopioidaan. **Plagiointi = hylkäys.** |
| 20 p | Aineiston käyttö summittaista. Viittaukset kömpelöitä. |
| 30 p | Aineistoja käytetään, mutta dialogi on heikkoa – enemmän referointia kuin analysointia. |
| 40 p | **Aktiivinen vuoropuhelu** aineistojen kanssa. Aineisto tukee näkökulmaa. |
| 50–60 p | Aineistojen käyttö on **luontevaa, analyyttistä ja omaperäistä**. Kokelas vie aineiston pidemmälle kuin mitä se suoraan sanoo. |

### Sensorin vinkki:
> *"Aineistoviittausten pitää olla kunnossa. Jos niitä ei ole tai ne ovat epämääräisiä, lukija ei tiedä, milloin sinä olet äänessä ja mikä tulee aineistosta. Plagiointi on vakava virhe."*

### Aineiston käytön tasovertailu:

```
 ❌ HUONO (referointi):
 "Aineistossa B sanotaan, että tekoäly voi korvata ihmisen työssä.
  Aineistossa C sanotaan, että tämä on ongelma."

 ⚠️ OK (maininta):
 "Tekoäly voi korvata ihmisen työssä (aineisto B), mikä herättää
  eettisiä kysymyksiä."

 ✅ HYVÄ (integrointi):
 "Kuten toimittaja Virtanen osoittaa, automaatio uhkaa erityisesti
  rutiininomaisia ammatteja. Tämä herättää kysymyksen siitä, kenen
  vastuulla on uudelleenkoulutus."

 🌟 ERINOMAINEN (dialogi):
 "Virtasen näkemys automaation uhasta on perusteltu, mutta se
  sivuuttaa teknologian luoman uuden työn. Tutkija Mäkelä (aineisto D)
  tarjoaa tasapainoisemman kuvan osoittaessaan, että jokainen
  kadonnut työpaikka on historiallisesti synnyttänyt 1,3 uutta.
  Molemmat näkökulmat paljastavat kuitenkin saman perusongelman:
  muutos on nopeampaa kuin yhteiskunnan kyky sopeutua."
```

## Linssi 5: KOKONAISKUVA – "Millainen kirjoittaja tämä on?"

### Mitä sensori kysyy itseltään:

```
┌─────────────────────────────────────────────────────┐
│  * Onko tekstissä KIRJOITTAJAN OMA ÄÄNI?            │
│  * Tuntuuko teksti KOKONAISUUDELTA?                 │
│  * Osoittaako teksti KYKYÄ AJATELLA ja ANALYSOIDA?  │
│  * Onko teksti KIINNOSTAVAA lukea?                  │
│  * Onko teksti OIKEAN MITTAINEN (~5000–6000 m.)?    │
└─────────────────────────────────────────────────────┘
```

> **Tämä on tärkein linssi.** Sensori muodostaa kokonaiskuvan kaikista osa-alueista yhdessä. Hyvä kokonaiskuva voi kompensoida yksittäisiä heikkouksia.

### Sensorin vinkki:
> *"Kokeeseen kannattaa mennä kuin urheilija tärkeään taistoon. Sillä fiiliksellä, että nyt menen tekemään oman parhaan suoritukseni."*
>
> *"Persoonallinen ääni on parempi kuin täysin neutraali teksti."*
>
> *"Kärkkäys sopii kantaa ottavaan tekstiin, kunhan perustelut ovat kunnossa."*

Käy tämä lista läpi kuin sensori lukisi tekstiäsi ensimmäistä kertaa:

```

│                    SENSORIN CHECK                            │
│                                                              │
│  ENNEN KAIKKEA:                                              │
│  □ Otsikko on olemassa ja kertoo näkökulman                  │
│  □ Viittaan vähintään 2 aineistoon SELKEÄSTI                 │
│  □ Teksti vastaa VALITTUUN AIHEESEEN (ei sivuraiteille)      │
│                                                              │
│  NÄKÖKULMA:                                                  │
│  □ Lukija tietää ensimmäisen kappaleen jälkeen,              │
│    mistä kulmasta kirjoitan                                  │
│  □ Kannanottoni on selkeä (en jää "aidalle")                 │
│  □ Käsittelen myös vastanäkökulmaa                           │
│                                                              │
│  RAKENNE:                                                    │
│  □ Johdanto esittelee aiheen ja näkökulman                   │
│  □ Jokainen kappale käsittelee YHTÄ ajatusta                 │
│  □ Kappaleet kytkeytyvät toisiinsa (sidossanat)              │
│  □ Lopetus kokoaa eikä tuo uutta                             │
│  □ Teksti on ~5000–6000 merkkiä (ei 9000+)                   │
│                                                              │
│  AINEISTOT:                                                  │
│  □ Aineistoviittaukset erottuvat omasta tekstistä            │
│  □ En vain referoi – analysoin ja tulkitsen                  │
│  □ Aineisto TUKEE argumenttiani (ei irrallinen)              │
│                                                              │
│  KIELI:                                                      │
│  □ Kirjoitan kirjakieltä (ei "mä", "sit", "niinku")          │
│  □ Virkkeet ovat eri mittaisia (ei pelkkiä lyhyitä/pitkiä)   │
│  □ Sanavalinnat ovat tarkkoja (ei "asia", "juttu", "homma")  │
│  □ Pilkut ovat kohdallaan (erityisesti sivulauseiden edessä) │
│  □ Yhdyssanat ovat yhdyssanoja                               │
│                                                              │
│  VIIMEINEN LUKEMINEN:                                        │
│  □ Luen tekstin mielessäni ääneen                            │
│  □ Jokainen virke on ymmärrettävä                            │
│  □ Otsikko vastaa todellista sisältöä                        │

│                                                              │

## 3 nopeinta tapaa nousta tasolta toiselle

### 20 p → 30 p: "Saa rakenteen kuntoon"
```
  Ongelma:  Teksti rönsyilee, ei kappaleiden välistä logiikkaa
  Ratkaisu: 1) Kirjoita ENSIN kappaleiden otsikot (vaikka piilotetut)
            2) Yksi kappale = yksi ajatus
            3) Lisää sidossanoja: "Toisaalta", "Sen sijaan", "Lisäksi"
```

### 30 p → 40 p: "Syvennä pohdintaa"
```
  Ongelma:  Pinnallista, yksipuolista, "koulukirjoittamista"
  Ratkaisu: 1) Kysy jokaisesta väitteestä: MIKSI? ja ENTÄ JOS EI?
            2) Tuo vastanäkökulma ja vastaa siihen
            3) Käytä aineistoja dialogin välineenä, älä referoi
```

### 40 p → 50+ p: "Löydä oma ääni"
```
  Ongelma:  Hyvä mutta "geneerinen", ei erotu joukosta
  Ratkaisu: 1) Aloita kiinnostavasti (ei "Tässä esseessä käsittelen...")
            2) Käytä konkreettisia esimerkkejä, omakohtaisuutta
            3) Tee yllättäviä kytkentöjä aineistojen välillä
            4) Pääty lopetukseen, joka jättää lukijalle ajateltavaa
```

║                                                      ║
║   Näkökulma  → Rajaa ja ota kanta.                   ║

║   Aineistot  → Vähintään 2, integroituna.            ║
║                                                      ║
║   Jos kaikki 5 on kunnossa,                          ║
║   sensori on tyytyväinen.                            ║
║                                                      ║
╚══════════════════════════════════════════════════════╝
```
