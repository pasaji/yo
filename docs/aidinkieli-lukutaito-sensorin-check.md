# Sensorin silmin – Lukutaidon kokeen arviointi

> Lukutaidon kokeessa sensori ei arvioi kirjoittamistasi vaan **lukemistasi** –
> kykyäsi eritellä, tulkita ja arvioida tekstejä. Tämä dokumentti näyttää,
> mitä sensori etsii vastauksestasi.

---

## Sensorin 3 lukutaidon tasoa: ETK-malli

Sensori arvioi vastauksesi kolmen **lukutaidon tason** läpi. Muista **ETK**:

```
╔══════════════════════════════════════════════════════════════╗
║                                                              ║
║     E  T  K                                                  ║
║     │  │  │                                                  ║
║     │  │  └───── K ulttuurinen lukutaito                     ║
║     │  └──────── T ulkitseva (kriittinen) lukutaito          ║
║     └─────────── E rittelevä lukutaito                       ║
║                                                              ║
║                                                              ║
║     Erittelevä   = MITÄ tekstissä on? (havainnot)            ║
║     Tulkitseva   = MIKSI ja MITEN? (päätelmät)               ║
║     Kulttuurinen = MISSÄ KONTEKSTISSA? (yhteydet)            ║
║                                                              ║
║                                                              ║
║     Pisteisiin vaikuttaa:                                    ║
║                                                              ║
║     Tyydyttävä  = pelkkää erittelyä                          ║
║     Hyvä        = erittelyä + kriittistä lukutaitoa          ║
║     Erinomainen = erittelyä + kriittistä + kulttuurista      ║
║                                                              ║
╚══════════════════════════════════════════════════════════════╝
```

**Kulttuurinen lukutaito erottaa hyvän eriomaisesta.** Se tarkoittaa, että osaat sijoittaa tekstin laajempaan kontekstiin.

## Taso 1: ERITTELEVÄ lukutaito – "Mitä tekstissä on?"

### Mitä sensori etsii:

```
┌─────────────────────────────────────────────────────┐
│  * Tekeekö kokelas HAVAINTOJA tekstin keinoista?    │
│  * NIMEÄÄKÖ hän keinot oikeilla käsitteillä?        │
│  * Perustaako hän havainnot TEKSTIVIITTEISIIN?      │
│  * Onko havaintoja RIITTÄVÄSTI ja monipuolisesti?   │
└─────────────────────────────────────────────────────┘
```

### Mitä tämä tarkoittaa käytännössä:

**Asiatekstissä** (Osa 1):
- Retoriset keinot: eetos, paatos, logos
- Kielen keinot: toisto, vastakkainasettelu, ironia, metafora
- Rakenteen piirteet: otsikko, aloitus, kappalejako
- Näkökulma: kenen silmin, kenen ääni

**Kaunokirjallisuudessa** (Osa 2):
- Kerronnan keinot: kertoja, fokalisaatio, aikarakenne
- Hahmot: luonne, kehitys, suhteet
- Kieli: kuvakieli, symbolit, sävy
- Miljöö: aika, paikka, tunnelma
- Rakenne: juoni, käänteet, lopetus

### Sensorin vinkki:
> *"Retorisia keinoja kannattaa nostaa vastauksissa esiin siltä osin, kun ne tukevat tehtävänannon mukaisia päätelmiä. Mekaaninen retoristen keinojen erittely ei kuulu tehtävänantoon."*

### Tasovertailu:

```
 ❌ HUONO (luettelointi):
 "Tekstissä käytetään metaforaa, toistoa ja ironiaa."

 ⚠️ OK (havainto):
 "Tekstissä käytetään toistoa, kun puhuja sanoo kolmesti
  'minä pelastin teidät'. Toisto korostaa puhujan itsevarmuutta."

 ✅ HYVÄ (havainto + päätelmä):
 "Kolumni rakentaa Marinista itsekeskeistä kuvaa toiston
  avulla: 'minä pelastin teidät' toistuu kolmesti ikään kuin
  Marin ei pystyisi puhumaan mistään muusta kuin itsestään.
  Tämä tukee kolumnin satiirista sävyä."
```

## Taso 2: TULKITSEVA (kriittinen) lukutaito – "Miksi ja miten?"

### Mitä sensori etsii:

```
┌─────────────────────────────────────────────────────┐
│  * Tekeekö kokelas PÄÄTELMIÄ havainnoistaan?        │
│  * Ymmärtääkö hän tekstin TAVOITTEEN?               │
│  * Tunnistaako hän TEKSTILAJIN ja sen konventiot?   │
│  * Yhdistääkö hän keinot tekstin KOKONAISUUTEEN?    │
│  * Tunnistaako hän sävyjä: ironian, sarkasmin?      │
│  * Esittääkö hän TULKINNAN, ei vain kuvailua?       │
└─────────────────────────────────────────────────────┘
```

### Havainto vs. päätelmä vs. tulkinta:

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   HAVAINTO    "Novellissa käytetään minäkertojaa."          │
│       │                                                     │
│       ▼                                                     │
│   PÄÄTELMÄ   "Minäkertoja tekee lukijasta osallisen         │
│               tapahtumiin ja luo epävarmuutta, koska        │
│               kertoja voi olla epäluotettava."              │
│       │                                                     │
│       ▼                                                     │
│   TULKINTA   "Novellin ydinteema on kuoleman                │
│               vääjäämättömyys: vaikka hahmot lukitsevat     │
│               itsensä kartanoon, Varjo eli kuolema          │
│               saapuu silti. Minäkertojan kuolemanjälkeinen  │
│               perspektiivi korostaa tätä – kukaan ei        │
│               lopulta paennut."                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘

  Erittelevä lukutaito = havainnot
  Kriittinen lukutaito = päätelmät
  Erinomainen vastaus  = kokonaistulkinta
```

### Sensorin vinkki:
> *"Kaunokirjallisuudessa harvoin on yhtä oikeaa tulkintaa. Sensorit ovat tästä hyvin tietoisia."*
>
> *"Mitään 'oikeaa' tulkintaa ei ole, kun on kaunokirjallisuudesta puhe. Vastauksessa voi päätyä monenlaisiin, yhtä hyviin tulkintoihin, kunhan ne vain pohjautuvat aineistoon."*
>
> *"Koesuorituksen tulee osoittaa kokelaan omaa lukemista, ei toistaa mallivastausta."*

## Taso 3: KULTTUURINEN lukutaito – "Missä kontekstissa?"

### Mitä sensori etsii:

```
┌─────────────────────────────────────────────────────┐
│  * Sijoittaako kokelas tekstin AIKAANSA?            │
│  * Tunnistaako hän GENREN ja sen perinteen?         │
│  * Huomaako hän INTERTEKSTUAALISET viittaukset?     │
│  * Yhdistääkö hän tekstin LAAJEMPAAN kontekstiin?   │
│  * Ymmärtääkö hän julkaisukanavan merkityksen?      │
│  * Tunnistaako hän yhteiskunnallisia kytkentöjä?    │
└─────────────────────────────────────────────────────┘
```

### Tämä on "bonustaso" – se erottaa hyvän erinomaisesta:

```
 HYVÄ vastaus (erittelevä + kriittinen):
 "Kolumni ivaa Marinia liioittelun ja sarkasmin avulla.
  Se rakentaa Marinista itsekeskeistä kuvaa."

 ERINOMAINEN vastaus (+ kulttuurinen):
 "Kolumni ivaa Marinia liioittelun ja sarkasmin avulla.
  Iltapäivälehden kolumni genrenä sallii kärjistykset,
  ja Lehtolan teksti jatkaa suomalaisen poliittisen
  satiirin perinnettä. Kolumni kommentoi samalla median
  tapaa rakentaa poliitikkojen julkisuuskuvia:
  henkilökuva ylistää nousevaa tähteä, kolumni pudottaa
  hänet jalustalta."
```

### Mitä "kulttuurinen lukutaito" tarkoittaa eri tehtävätyypeissä:

| Osa 1 (asiatekstit) | Osa 2 (kaunokirjallisuus) |
|---------------------|--------------------------|
| Julkaisukanava ja sen merkitys (HS vs. IS) | Kirjallinen perinne ja genre (kauhu, romantiikka) |
| Media-analyysi: miten media rakentaa kuvia | Intertekstuaalisuus (Raamattu, mytologia) |
| Poliittinen/yhteiskunnallinen konteksti | Kirjailijan aikakausi ja suuntaus |
| Tekstilajien konventiot (kolumni, henkilökuva) | Symboliikan kulttuuriset merkitykset |
| Kohderyhmä ja viestinnän trendit | Teksti suhteessa muuhun kirjallisuuteen |

### Sensorin vinkki:
> *"Kriittistä lukutaitoa osoittaa keinojen ja päätelmien yhdistäminen tekstien tavoitteisiin sekä tekstilajien ja julkaisukanavien eroihin."*
>
> *"Kulttuurista lukutaitoa osoittaa esimerkiksi päätelmien suhteuttaminen median yleisempiin toimintatapoihin ja laajemmin yhteiskunnalliseen keskusteluun."*

## Osa 1 vs. Osa 2: Eri linssi, sama logiikka

### Osa 1: Asiatekstit – sensori haluaa nähdä:

```
┌──────────────────────────────────────────────────────────┐
│  ASIATEKSTIN ANALYYSI – sensorin tarkistuslista          │
│                                                          │
│  □ Vastaa tehtävänantoon (ei sivuraiteille)              │
│  □ Tekee havaintoja KEINOISTA (ei vain sisällöstä)       │
│  □ Yhdistää keinot tekstin TAVOITTEESEEN                 │
│  □ Tunnistaa TEKSTILAJIN ja sen konventiot               │
│  □ Käyttää käsitteitä: retoriikka, argumentaatio,        │
│    rekisteri, kohderyhmä, genre                          │
│  □ Vertailuu tehtävissä: EROja JA yhtäläisyyksiä         │
│  □ Päätelmiä, ei pelkkää referointia                     │
│  □ Kriittistä otetta: kenen etu, kenen ääni?             │
│                                                          │
│  BONUS (kulttuurinen lukutaito):                         │
│  □ Julkaisukanavan merkitys (mistä lehdestä/mediasta?)   │
│  □ Yhteiskunnallinen konteksti ja ajankohtaisuus         │
│  □ Median toimintatapojen tunnistaminen                  │
└──────────────────────────────────────────────────────────┘
```

### Osa 2: Kaunokirjallisuus – sensori haluaa nähdä:

```
┌──────────────────────────────────────────────────────────┐
│  KAUNOKIRJALLISUUDEN ANALYYSI – sensorin tarkistuslista  │
│                                                          │
│  □ Vastaa tehtävänantoon (analyysi + tulkinta)           │
│  □ Tekee havaintoja KEINOISTA: kerronta, kieli,          │
│    hahmot, miljöö, rakenne                               │
│  □ Käyttää KAUNOKIRJALLISUUDEN käsitteitä                │
│    (ei argumentaatioanalyysia!)                          │
│  □ Perustelee havainnot TEKSTIVIITTEIN                   │
│  □ Tekee KOKONAISTULKINNAN (mistä teksti kertoo?)        │
│  □ Ei vain referoi juonta                                │
│  □ Yhdistää keinot tulkintaan (keino → merkitys)         │
│                                                          │
│  BONUS (kulttuurinen lukutaito):                         │
│  □ Genreperinne (kauhu, romantiikka, realismi)           │
│  □ Intertekstuaalisuus (viittaukset muihin teksteihin)   │
│  □ Kirjailijan aikakausi ja suuntaus                     │
│  □ Symbolien kulttuurinen merkitys                       │
└──────────────────────────────────────────────────────────┘
```

## Sensorien useimmin toistamat neuvot

Koottu suoraan YLE Abitreenien Q&A-sessioista (S2025):

### 1. "Perustelut ratkaisevat"
```
  Sensori toisti tämän kymmeniä kertoja. Jos tulkintasi on
  erilainen kuin HVP:ssä, se voi silti olla oikein –
  KUNHAN PERUSTELLAAN TEKSTISTÄ.

  "Kaunokirjallisuudessa harvoin on yhtä oikeaa tulkintaa,
   ja sensorit ovat tästä hyvin tietoisia."
```

### 2. "HVP ei ole mallivastaus"
```
  Hyvän vastauksen piirteet (HVP) on lista MAHDOLLISIA
  havaintoja. Erinomainenkaan vastaus ei tarvitse kaikkia
  niitä. Vastauksessa voi myös olla asioita, joita HVP:ssä
  ei mainita.

  "Vastauksessa voi tehdä sellaisiakin hyviä havaintoja,
   joita HVP:ssa ei mainita."
```

### 3. "Vastaus arvioidaan KOKONAISUUTENA"
```
  Yksittäinen virhe (väärä käsite, aikakauden ajoitusvirhe,
  puuttuva otsikko) EI romuta vastausta.

  "Pisteitä ei anneta merkkimäärän vaan sisällön perusteella."
  "Vastaukset arvioidaan aina kokonaisuutena eikä väärien
   käsitteiden käytöstä tehdä automaattisia pistevähennyksiä."
```

### 4. "Tulkinta ≠ referointi"
```
  Juonen kuvaaminen ei ole analyysia. Sensori haluaa
  nähdä MIKSI-kysymyksiä:

  ❌ "Novellissa hahmot juovat viiniä ja sitten varjo tulee."
  ✅ "Viinin ja juhlimisen kuvaus korostaa hahmojen
      yritystä paeta kuolemaa – turhaan, sillä Varjo
      saapuu silti."
```

### 5. "Käsitteet ovat työkalu, eivät itsetarkoitus"
```
  Käytä oikeita käsitteitä, mutta älä luettele niitä.
  Käsitteen pitää PALVELLA analyysia:

  ❌ "Tekstissä on metafora, personifikaatio ja anafora."
  ✅ "Anafora 'Elää, elää, elää!' luo ekstaattisen
      tunnelman ja korostaa elämänhalua."

  Asiatekstissä: retorisia keinoja, argumentaatiota
  Kaunokirjallisuudessa: kerrontaa, kuvakieltä, hahmoja
  → ÄLÄ sekoita näitä keskenään!
```

## Vastaamisen rakenne

### 30 pisteen tehtävä (~4500 merkkiä):

```
┌──────────────────────────────────────────────────────────┐
│  KAPPALE 1: Johdanto                                     │
│  • Mistä aineistosta on kyse? (1–2 lausetta)             │
│  • Mitä aiot analysoida? (= tehtävänanto omin sanoin)    │
├──────────────────────────────────────────────────────────┤
│  KAPPALEET 2–4: Analyysi                                 │
│  • Kappale = yksi havainto/näkökulma                     │
│  • Jokaisessa: havainto → tekstiviite → päätelmä         │
│  • Käytä käsitteitä luontevasti                          │
├──────────────────────────────────────────────────────────┤
│  KAPPALE 5: Kokonaistulkinta / Yhteenveto                │
│  • Mitä teksti kertoo syvemmällä tasolla?                │
│  • Vertailutehtävissä: kokoa erot ja yhteydet            │
│  • Kulttuurinen konteksti (jos osaat)                    │
└──────────────────────────────────────────────────────────┘
```

### Kaksiosainen tehtävä (12 p + 18 p):

```
  OSA A (12 p, ~2000–2500 merkkiä): ERITTELE
  ─────────────────────────────────────────────
  • Konkreettisia havaintoja keinoista
  • Nimeä keinot käsitteillä
  • Perustele lyhyillä tekstiviitteillä
  • Esim: "Millaisia oletuksia video tekee?"
         → Listaa oletukset ja perustele

  OSA B (18 p, ~2000–2500 merkkiä): TULKITSE
  ─────────────────────────────────────────────
  • Syvemmälle: MIKSI nämä keinot, MITÄ ne merkitsevät?
  • Yhdistä tavoitteeseen ja kontekstiin
  • ÄLÄ toista A-osan sisältöä – vie pidemmälle
  • Esim: "Miten video hyödyntää eri tekstien piirteitä?"
         → Tunnista genret ja selitä miksi ne on valittu
```

### Sensorin vinkki:
> *"Merkkimäärä on suositus, ei tiukka kriteeri. Jos saa sanottavansa sanotuksi alle vaaditun merkkimäärän, on vastaus onnistuneesti rajattu. Toki runsaasti alamittaisista vastauksista usein puuttuu jotain."*

## Yleisimmät virheet – sensorin silmin

```
 VIRHE                           MIKSI SE HAITTAA
 ──────────────────────────────────────────────────────
 Juonen referointi               Ei osoita lukutaitoa, vain muistia.
 kaunokirjallisuudessa           → Analysoi KEINOJA, älä kerro juonta.

 Argumentaatioanalyysi           Väärä työkalu. Novelli ei argumentoi.
 kaunokirjallisuudessa           → Käytä kerronnan, kielen, hahmojen
                                   käsitteitä.

 Keinojen luettelointi           Mekaanista, ei osoita ymmärrystä.
 ilman päätelmiä                 → Joka havainnosta päätelmä:
                                   "keino X → vaikutus Y"

 Tehtävänannon ohittaminen       Sensori arvioi tehtävänannon mukaan.
                                 → Lue tehtävänanto 3 kertaa!

 Tulkinnan puuttuminen           Ilman tulkintaa max tyydyttävä taso.
 kokonaan                        → Lopeta aina kokonaistulkintaan.

 "HVP sanoo..." -ajattelu        Sensori ei odota mallivastausta.
                                 → Tee OMA perusteltu tulkinta.
```

## 3 nopeinta tapaa nousta tasolta toiselle

### Tyydyttävä → Hyvä: "Älä referoi, analysoi"
```
  Ongelma:  Kerrot mitä tekstissä tapahtuu tai sanotaan
  Ratkaisu: Kysy jokaisesta havainnosta: MIKSI ja MITÄ VARTEN?
            → "Metafora 'elämä on matka' korostaa elämän
               ohimenevyyttä ja kehottaa nauttimaan hetkestä."
```

### Hyvä → Erittäin hyvä: "Keinot → kokonaisuus"
```
  Ongelma:  Havainnot ovat irrallisia, eivät muodosta kokonaisuutta
  Ratkaisu: Päätä vastauksesi KOKONAISTULKINTAAN, joka sitoo
            kaikki havaintosi yhteen.
            → "Novellin kaikki keinot – minäkertoja, synkkä miljöö,
               yliluonnolliset hahmot – rakentavat yhdessä
               tulkinnan kuoleman vääjäämättömyydestä."
```

### Erittäin hyvä → Erinomainen: "Konteksti ja yhteydet"
```
  Ongelma:  Analyysi on tarkka mutta pysyy tekstin sisällä
  Ratkaisu: Sijoita teksti kontekstiin:
            • Mikä genre? Miten se noudattaa/rikkoo konventioita?
            • Mikä aikakausi? Miten se näkyy?
            • Mihin muihin teksteihin se viittaa?
            → "Poen novelli jatkaa goottilaisen kauhun perinnettä
               mutta siirtää miljöön keskiaikaisista linnoista
               antiikin Kreikkaan, mikä luo oman ainutlaatuisen
               jännitteensä."
```
